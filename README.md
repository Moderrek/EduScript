## Features

* Polish syntax
* Variables
* Builtin functions (wypisz)

## Required

* Java 17
* Knowledge of the Polish language :)

# Examples

### EduScript

* `stala` = `const`
* `zmien` = `var`
* `wypisz` = `print`
* *promien* means *radius*
* *pole* means *area*

```es
zmien promien = 5;
zmien pole = promien * promien * pi;
wypisz("Pole wynosi " + pole);
```

### Python

```py
promien = 5
pole = promien * promien * pi;
print(f"Pole wynosi {pole}")
```
